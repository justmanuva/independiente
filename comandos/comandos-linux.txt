- pwd: Ver ruta en la que estoy
- touch (+nombre): Crear archivos
- nano (+nombre): Crear archivos (más detallado)
	-(-l para numeros de línea)
- mv (+nombre): Renombrar archivo (o carpeta?)
	- (+ruta/nombre) (ruta): Mueve el archivo a la ruta de la derecha
- rm(dir)(+nombre): Borrar archivo(carpeta)
	- (-r para por si tiene archivos dentro)
	- (-f para forzar la eliminación)
- sudo dpkg --list: Listar todos los Paquetes
- sudo dpkg --list | grep (nombre): Listar por nombre
- sudo apt-get --purge remove (nombrePaquete): Borrar paquete
- ln -s (rutaOrigenArchivo) (rutaAccesoArchivo): Crea acceso directo
